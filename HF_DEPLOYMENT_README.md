## 3. How to Push Updates
To push new code changes to the Hugging Face Space:

1. **Navigate to the directory:**
   ```bash
   cd hf_space
   ```

2. **Sync changes from the main project:**
   Ensure the `src/` files in `hf_space/` match your latest development code.

3. **Stage and Commit:**
   ```bash
   git add .
   git commit -m "Description of your changes"
   ```

4. **Push to Hugging Face:**
   ```bash
   git push origin main
   ```

## 4. Key Deployment Files

## 5. Endpoints Available in Production
Once pushed, the API is accessible at:
`https://lulc-recognition-lulc-backend.hf.space`

- **Standard Inference:** `POST /predict` (multipart/form-data)
- **Mobile Inference:** `POST /predict_mobile` (JSON + Base64) - *Added for robust mobile support*
- **Health Check:** `GET /health`

## 6. Monitoring & Logs
If the API is "rejecting" requests or failing:
1. Go to the **Hugging Face Space UI**.
2. Click the **Logs** tab.
3. Look for `INFO:lulc-api` tags. I have added middleware that logs the **User-Agent** and **Headers** of every incoming request to help debug connection issues from mobile apps.
